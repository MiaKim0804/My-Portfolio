# My-Portfolio

## Description

This is my portfolio website. It is built with Next.js, TailwindCSS, and TypeScript.

## Installation

```bash
npm install
npx tailwindcss init -p
npx create-next-app 
```

## Usage

```bash
npm run dev
```
